小软件

Quickly build an effective pricing table for your potential customers with this Bootstrap example. It’s built with default Bootstrap components and utilities with little customization.

PDM

  • 环境版本：Microsoft .NET Framework 4.0
  • 程序功能：支持条码二维码打印机进行编辑排版的软件，支持文字、图片、条码、二维码、直线，能自由拖拉，删除，并可保存为模版。

RSS编码解码器

  • 环境版本：Microsoft .NET Framework 4.0
  • 程序功能：实现RSS的读取和生成，支持RSS2.0和Atom格式。

照片批量压缩工具

  • 环境版本：Microsoft .NET Framework 4.0
  • 程序功能：对大照片进行批量压缩和格式转换，该工具界面精美，速度快。只支持win7/Vista/win8系统。

C#实体类生成器

  • 环境版本：Microsoft .NET Framework 4.0
  • 程序功能：用于直接把数据库表生成实体类。已支持mysql、oracle数据库。

C#Excel导入导出组件

  • 环境版本：Microsoft .NET Framework 4.0
  • 程序功能：实现Excel数据的导入导出。

C#代码执行者

  • 环境版本：Microsoft .NET Framework 2.0
  • 程序功能：在TextBox中输入一段C＃代码，点击按钮即可执行该代码。
